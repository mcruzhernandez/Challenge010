# BancoCuscatlanChallenge (2 revision)

Este proyecto fue realizado utilizando Xcode y SwiftUI

El proyecto utiliza las librerías externas:
- Alamofire
- Nuke

Las libraries se agregaron utilizando Swift Package Manager, ya que Cocoapods no contiene las ultimas actualizaciones de Nuke.

El proyecto puede visualizarse utilizando cualquier simulador o dispositivo físico.
